# 쇠막대기 2024-10-28

## 풀이
```python
galho = input()
stack = []
answer = 0

for i in range(len(galho)):
    if galho[i] == '(':
        stack.append('(')
    else:
        if galho[i-1] == '(':
            stack.pop()
            answer += len(stack)
        else:
            stack.pop()
            answer += 1
print(answer)
```
