# 백준 2579번 - 계단 오르기

```python
n = int(input())
score = [0] + [int(input()) for _ in range(n) ]
dp = [0] * (n + 1)

# 2보다 작으면 다 밟는게 최대 점수
if len(score) <= 2:
    print(sum(score))

else:
    dp[1] = score[1]
    dp[2] = score[1] + score[2]
    for i in range(3, n+1):
        # 1. i-3번째 계단에서 i-1번째와 i번째 계단을 연속으로 밟는 경우
        # 2. i-2번째 계단에서 i번째 계단을 밟는 경우
        dp[i] = max(dp[i-3] + score[i-1] + score[i], dp[i-2] + score[i])
print(dp[n])
```
